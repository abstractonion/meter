# Typography Craft

A **meter** discipline module — the verbal-visual layer. Type is voice made visible; design traditions from manuscript tradition through Swiss modernism inform how we set text in product UI.

## Core Philosophy

### Type is interface, not decoration

Users read to act. Every size step, weight change, and line length either aids scanning or fights it. Typography that looks "designed" but reads poorly has failed.

### Optical truth beats mathematical equality

Equal point sizes across families look unequal. Kerning pairs, ascender height, and x-height shift perceived weight. Trust the eye, then measure.

### Rhythm compounds

Consistent vertical rhythm (baseline grid or modular scale) makes dense UI feel calm. Broken rhythm feels like a draft.
